# SEO/GEO Field Model

Every public collection gets a first-class `seo` group and `geo` group. GEO answer summaries are not buried in rich text.

## SEO Group

- `metaTitle`
- `metaDescription`
- `canonicalUrl`
- `ogTitle`
- `ogDescription`
- `ogImage`
- `robots`
- `schemaType`
- `llmsTxt`

## GEO Group

- `answerSummary`
- `keyQuestions[].question`
- `keyQuestions[].shortAnswer`
- `aiSearchIntent`
- `citationSummary`
- `confidenceNotes`

## Homepage Questions

The homepage seed includes the required GEO questions: what VIAIVE is, how it helps plan luxury travel, when to use a luxury travel advisor, what trip types it supports, and how hotels/destinations are chosen.
