# Design Token Implementation

Source tokens are preserved at `docs/figma/source/docs/design-tokens.json`.

Runtime CSS tokens are implemented in `src/styles/tokens.css` with the same core Stone & Paper values:

- `--paper`
- `--paper-alt`
- `--stone`
- `--stone-alt`
- `--stone-mid`
- `--gold`
- `--gold-soft`
- `--shadow-paper`
- `--shadow-gold`
- `--shadow-hero`

No arbitrary alternate palette was introduced.
